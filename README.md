# App-Para-Salvar-Receita

## Sobre

Fiz esse app com o objetivo de aprender e praticar o manuseio de banco de dados em uma aplicação mobile. 

## Recursos

- Adicionar Receitas
- Editar Receitas
- Excluir Receitas
- Ver todas as receitas em uma lista e lê-las com mais detalhes ao clicar
